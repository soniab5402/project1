# 🎬 FlixStream

FlixStream is a web based movie recommendation web app built developed using Flask.it allows user to log in and get movie suggestions based on genre.The application features a simple and clean interface inspred by netflix,making it easy for the users to explore movies.This project demonstrates backened development using flask,basic frontend design with HTML/CSS.

## 🚀 Features
- User Login System
- Movie Recommendations
- Genre Filtering

## 🛠 Tech Stack
- Python (Flask)
- HTML, CSS

## ▶️ Run Project
python app.py
